# Portfolio Website

Static portfolio website built with plain HTML, CSS, and JavaScript.

## Open the site

Open `index.html` in any browser.

## Customize

- Update your name and intro in `index.html`
- Replace the sample project cards in `index.html`
- Change the email and social links in `index.html`
- Adjust colors and spacing in `styles.css`

## Files

- `index.html` for page structure and content
- `styles.css` for layout, colors, and responsive design
- `script.js` for the footer year and reveal-on-scroll animation
